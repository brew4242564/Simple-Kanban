# About
Simple kanban is a trello copy made for educational purposes using Javascript, HTML and CSS.

## What can I do in this app?
* Create new Lists
* Add cards
* Edit cards and lists
* Delete cards and lists
* Save the table in LocalStorage

## Upcoming updates
* Animations
* Data export and import
